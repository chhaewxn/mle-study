# 단어집 📓
| **용어** | **정의** |
| --- | --- |
| Throughput (처리량) | 특정 기간 내에 처리된 쿼리 또는 요청의 수 (처리량) |
| Latency (레이턴시) | 요청이 시작된 후 응답이 돌아오기까지 걸리는 순수 처리 지연 시간 |
| Response Time (응답 시간) | 레이턴시에 네트워크 지연, 대기열 지연 등 기타 요소를 모두 포함한 최종 시간 |
| Off-the-shelf (기성) | 기성품이라는 뜻으로, 커스터마이징 없이 바로 사용 가능한 라이브러리/모델/솔루션 |
| Ticket (고객 지원 티켓) | 고객이 회사에 문의하거나 문제를 제기할 때 생성되는 문서/요청서 같은 것. 영화관 티켓처럼 번호표를 받는다는 의미에서 유래  |
| Use Case (유스 케이스) | 어떤 기술을 ‘어디에, 어떻게 쓰는지’를 나타내는 구체적인 사용 시나리오. ML 유스 케이스의 경우 머신러닝을 써서 해결하는 구체적인 문제/사용 시나리오를 뜻함 |
